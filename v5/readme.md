# Lernziele

Ziel dieser Aufgabe ist es, weitere Funktionen für Treiberprogrammierer kennen zu lernen.
Dazu werden zuerst Funktionen, die im Kernelkontext laufen vorgestellt, sodass Sie Ihre Funktionen des Prozesskontextes entsprechend intelligent implementieren können.

## Folgende Themen werden erarbeitet:

- Funktionen im Kernel Kontext.
- Zugriffsmodi des Treibers.
